Task Manager API Documentation

/api/auth

POST /api/auth/login

Function: Authenticates a user with email and password. Sets an auth cookie on success.

Example Body:

{
  "email": "user@example.com",
  "password": "password123"
}

Example Response:

{
  "message": "Login successful",
  "name": "John Doe",
  "email": "user@example.com"
}

POST /api/auth/register

Function: Registers a new user with name, email, and password.

Example Body:

{
  "name": "John Doe",
  "email": "user@example.com",
  "password": "password123"
}

Example Response:

{
  "message": "User created successfully",
  "name": "John Doe",
  "email": "user@example.com"
}

POST /api/auth/logout

Function: Logs out the user by clearing the auth cookie.

Example Response:

{
  "message": "Logged out successfully"
}

/api/task

GET /api/task/:email

Function: Retrieves all tasks for a specific user by email.

Example Response:

{
  "tasks": [
    {
      "taskName": "Buy groceries",
      "color": "#f2d312",
      "status": false,
      "createdAt": "2025-04-13T12:00:00Z"
    }
  ]
}

POST /api/task/add

Function: Adds a new task to a user's task list.

Example Body:

{
  "email": "user@example.com",
  "data": {
    "taskName": "Finish homework",
    "color": "#ff0000"
  }
}

Example Response:

{
  "message": "Task Added"
}

POST /api/task/complete/:email

Function: Marks a task as completed for a given user.

Example Body:

{
  "id": "taskId123"
}

Example Response:

{
  "message": "Task marked as completed"
}

DELETE /api/task/delete/:email

Function: Deletes a task from a user's task list.

Example Body:

{
  "id": "taskId123"
}

Example Response:

{
  "message": "Task deleted successfully"
}
