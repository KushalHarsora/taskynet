Task Manager API Documentation

/api/auth

POST /api/auth/login

Function: Authenticates a user with email and password. Returns a JWT token.

Example Body (JSON):

{
  "email": "user@example.com",
  "password": "password123"
}

Example Response:

{
  "token": "jwt-token-here",
  "user": {
    "id": "userId",
    "email": "user@example.com"
  }
}

/api/tasks

GET /api/tasks

Function: Retrieves all tasks for the logged-in user. Requires JWT in headers.

Example Response:

[
  {
    "id": "task1",
    "title": "Buy groceries",
    "completed": false,
    "createdAt": "2025-04-13T12:00:00Z"
  }
]

POST /api/tasks

Function: Adds a new task for the user. Requires JWT.

Example Body:

{
  "title": "Finish assignment"
}

Example Response:

{
  "message": "Task created successfully"
}

PATCH /api/tasks/:id/complete

Function: Marks a task as completed. Requires JWT.

Example Request:

PATCH /api/tasks/12345/complete

Example Response:

{
  "message": "Task marked as completed"
}

DELETE /api/tasks/:id

Function: Deletes a task by ID. Requires JWT.

Example Request:

DELETE /api/tasks/12345

Example Response:

{
  "message": "Task deleted successfully"
}
